Calculadora em React
Este projeto implementa uma calculadora simples utilizando React. 
A calculadora suporta operações básicas como adição, subtração, multiplicação e divisão. 
Ela também permite limpar o display e utilizar ponto decimal.

Funcionalidades
A calculadora possui as seguintes funcionalidades:

- Exibir o valor atual no display.
- Adicionar dígitos (0-9) e ponto decimal.
- Realizar operações de adição, subtração, multiplicação e divisão.
- Limpar o display (botão "AC").
- Manter o estado das operações e valores intermediários.

Como Executar
1. Clone o repositório:

git clone https://github.com/seu-usuario/calculadora-react.git
cd calculadora-react

2. Instale as dependências:

npm install

3. Inicie o servidor de desenvolvimento:

npm start

4. Abra o navegador e acesse:

http://localhost:3000
